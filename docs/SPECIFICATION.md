# مواصفات موقع جمعية أصدقاء جامعة بيرزيت

## 1. المواصفات العامة للموقع (Global Styles)

| البند | الوصف |
|-------|--------|
| **اتجاه الموقع** | كامل الموقع يدعم RTL و LTR (حسب لغة العرض) |
| **نمط العرض** | Light Mode / Dark Mode مع زر تبديل واضح |
| **الهوية البصرية** | ألوان الجمعية الرسمية + لون مميز للإجراءات (Action Color) |
| **الاستجابة** | الموقع متجاوب بالكامل (Responsive Design) |

---

## 2. خريطة الموقع (Sitemap)

| القسم | المحتوى |
|-------|---------|
| **الرئيسية** | Home Page |
| **من نحن** | من نحن، الرؤية/الرسالة/الأهداف/القيم، فريقنا (مجلس الإدارة + الفريق التنفيذي) |
| **مشاريعنا** | كنعاني، تمكين، المظلات (تفاصيل كل مشروع في صفحة منفصلة) |
| **المنح الجامعية** | قائمة المنح → تفاصيل المنحة → نموذج التقديم الرقمي |
| **شركاء النجاح** | أفراد، شركات/مؤسسات |
| **الأخبار** | آخر الأخبار (صفحة لكل خبر) + إجراءات تفاعل |
| **تواصل معنا** | تبرع مباشر (في كل الصفحات)، طلب مساهمة تمكين، طلب انضمام للهيئة العامة |

---

## 3. هيكلية الصفحات (Page-by-Page Breakdown)

### 3.1 الصفحة الرئيسية (Home Page)

| القسم | المحتوى / المكونات |
|-------|---------------------|
| **Header** | شعار الجمعية + شريط التنقل الرئيسي + زر بارز "تبرع الآن" |
| **Hero** | صورة أو فيديو يعبر عن رسالة الجمعية + نص تعريفي قصير + زر CTA |
| **من نحن (نبذة)** | الرؤية، الرسالة، الأهداف + إحصائيات (إنفوجرافيك بسيط) |
| **أقسام المشاريع** | بطاقات (Cards) للمشاريع: كنعاني، تمكين، المظلات |
| **قصص النجاح** | مختارات من قصص المستفيدين (نص + صورة) |
| **شركاء النجاح** | شعارات الشركاء (أفراد وشركات) |
| **آخر الأخبار** | آخر 3 أخبار |
| **Footer** | خريطة الموقع، روابط التواصل الاجتماعي، معلومات الاتصال |

---

### 3.2 صفحات المشاريع (Projects Pages)

#### أ. مشروع كنعاني (Kan'ani)

| القسم | الوصف |
|-------|--------|
| **القسم العلوي** | فيديو تعريفي (Autoplay + Muted) + نبذة عن المشروع |
| **معرض الصور** | شبكة صور لمنتجات المشروع؛ كل صورة تفتح نافذة منبثقة (صورة + عنوان + وصف) |
| **زر التفاعل** | زر "زيارة المتجر" (CTA) في نهاية الصفحة |

**ملاحظات للتحليل:**  
- جدول/محتوى: فيديو واحد، نص نبذة، قائمة صور (صورة، عنوان، وصف)، رابط المتجر.

---

#### ب. مشروع تمكين (Tamkeen)

| البند | الوصف |
|-------|--------|
| **التصميم** | نظام "بطاقات الشراكة" (Partnership Cards) |
| **المحتوى لكل بطاقة** | اسم الجهة الداعمة، تاريخ البدء، عدد الطلبة المستفيدين |

**ملاحظات للتحليل:**  
- جدول شراكات تمكين: جهة داعمة، تاريخ بدء، عدد المستفيدين (+ أي حقول إضافية مثل الرابط أو الوصف).

---

#### ج. مشروع المظلات (Parasols)

| القسم | الوصف |
|-------|--------|
| **المحتوى النصي** | نص وصفي مختصر يشرح فكرة مساحات الإعلانات |
| **معرض الصور** | صور حسب المناطق الجغرافية: جامعة بيرزيت، بلدة بيرزيت، رام الله |
| **التفاعل** | Lightbox لتكبير الصور |

**ملاحظات للتحليل:**  
- جدول مناطق + جدول صور مرتبطة بمنطقة (أو tags). نص ثابت للوصف.

---

### 3.3 صفحات المنح (Scholarships Pages)

#### أ. قائمة المنح (Listing)

| العنصر | الوصف |
|--------|--------|
| **العرض** | بطاقات (Cards) للمنح المتاحة |
| **محتوى كل بطاقة** | صورة تعبيرية، عنوان المنحة، ملخص، فترة التقديم |
| **زر** | "تفاصيل / تقديم" |

#### ب. صفحة تفاصيل المنحة (Single Scholarship)

| القسم | الوصف |
|-------|--------|
| **المحتوى** | تفاصيل كاملة: التفاصيل، الشروط، الأوراق المطلوبة، التواريخ |
| **منطقة التحميل** | زر واضح لتحميل استمارة التقديم (PDF) |
| **نموذج التقديم الرقمي** | حقول: الاسم، رقم الهوية + حقل رفع ملفات (PDF/Word) + زر إرسال |

**ملاحظات للتحليل:**  
- جدول منح: عنوان، ملخص، صورة، فترة تقديم (من–إلى)، تفاصيل، شروط، أوراق مطلوبة، ملف PDF للاستمارة، إلخ.  
- جدول طلبات تقديم: مرتبط بمنحة، اسم مقدم الطلب، رقم هوية، ملف مرفق، تاريخ، حالة.

### 3.4 صفحات من نحن (About Us)

#### أ. صفحة التعريف

| القسم | الوصف |
|-------|--------|
| **الفيديو** | فيديو "قصة الجمعية" |
| **الأقسام النصية** | الرؤية، الرسالة، الأهداف، القيم (منظمة كنصوص/أقسام) |

#### ب. صفحة الفريق

| القسم | المحتوى |
|-------|---------|
| **الإدارة** | بطاقات: (صورة، الاسم، المسمى الوظيفي) — مجلس الإدارة / الفريق التنفيذي |
| **فريق العمل** | قائمة: الأسماء + المسميات الوظيفية (بدون صورة أو بصيغة أبسط) |

**ملاحظات للتحليل:**  
- محتوى ثابت أو جدول "صفحة من نحن" (فيديو، نصوص الرؤية/الرسالة/الأهداف/القيم).  
- جدول أعضاء الإدارة: صورة، اسم، مسمى، ترتيب، نوع (مجلس إدارة / تنفيذي).  
- جدول فريق العمل: اسم، مسمى وظيفي، ترتيب.

### 3.5 صفحات الأخبار وشركاء النجاح

#### أ. صفحة الأخبار

| البند | الوصف |
|-------|--------|
| **العرض** | بطاقات: (صورة/فيديو، عنوان، تفاصيل، تاريخ) |
| **الخصائص** | التصفية حسب التاريخ |
| **التفاصيل** | كل خبر في صفحة منفصلة (حسب الخريطة) |

**ملاحظات للتحليل:**  
- جدول أخبار: عنوان، ملخص، محتوى كامل، صورة/فيديو، تاريخ، قد يُضاف تصنيف أو وسوم.

#### ب. صفحة الشركاء

| البند | الوصف |
|-------|--------|
| **التقسيم** | فئتان: أفراد / شركات (ومؤسسات) |
| **العرض** | شعارات + أسماء بوضوح |

**ملاحظات للتحليل:**  
- جدول شركاء: اسم، شعار (صورة)، نوع (فرد / شركة)، ترتيب، رابط اختياري.

## 4. الوظائف والخصائص التقنية (Key Features)

| الوظيفة | الوصف |
|---------|--------|
| **أزرار التبرع** | زر التبرع ثابت (Sticky) أو متكرر في الصفحات الاستراتيجية |
| **نماذج التسجيل والتقديم** | ربط نماذج التقديم بقاعدة البيانات لتسجيل وتوظيف الطلبات |
| **إدارة العضوية العامة** | قسم لتحميل طلب الانضمام (PDF) + شرح سياسة العضوية والالتزامات المالية |
| **إدارة الملفات** | نظام رفع ملفات آمن لاستلام طلبات المنح من الطلاب (PDF/Word) |

## 5. ملخص الصفحات والجداول (للاستخدام في التحليل)

### المشاريع

| المشروع | نوع المحتوى الرئيسي | جداول/كيانات مقترحة |
|---------|---------------------|----------------------|
| **كنعاني** | فيديو، نبذة، معرض منتجات (صورة+عنوان+وصف)، رابط متجر | محتوى ثابت + معرض صور/منتجات |
| **تمكين** | بطاقات شراكة (جهة، تاريخ، عدد مستفيدين) | شراكات تمكين (قائمة سجلات) |
| **المظلات** | وصف قصير، معرض صور حسب المنطقة (3 مناطق) | مناطق + صور مرتبطة بمنطقة |

### المنح

| الصفحة | المحتوى | جداول مقترحة |
|--------|---------|---------------|
| قائمة المنح | بطاقات (صورة، عنوان، ملخص، فترة تقديم) | scholarships |
| تفاصيل + نموذج | تفاصيل، شروط، أوراق مطلوبة، PDF، نموذج (اسم، هوية، رفع ملف) | scholarship_applications، ملفات مرفقة |

### من نحن والفريق

| الصفحة | المحتوى | جداول مقترحة |
|--------|---------|---------------|
| التعريف | فيديو قصة الجمعية + الرؤية/الرسالة/الأهداف/القيم | about_page (أو محتوى ثابت) |
| الفريق | إدارة: بطاقات (صورة، اسم، مسمى)؛ فريق العمل: قائمة أسماء ومسميات | board_members، executive_team، staff_members |

### الأخبار والشركاء

| القسم | المحتوى | جداول مقترحة |
|-------|---------|---------------|
| الأخبار | بطاقات (صورة/فيديو، عنوان، تفاصيل، تاريخ) + تصفية حسب التاريخ | news |
| الشركاء | أفراد / شركات؛ شعارات + أسماء | partners (نوع: فرد/شركة) |
